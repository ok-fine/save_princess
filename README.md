# save_princess

一款闯关类的拯救公主小游戏

在解救公主的路上，会有蝙蝠怪的追击（部分蝙蝠怪有追击功能），你需要躲过蝙蝠怪，成功到达公主所在地，并将她带出门。
注意：救到公主后，如果再次遇到蝙蝠怪，则会将公主带回关押地。

每解救一次公主，可以开启下一个难度。蝙蝠数量会增加，蝙蝠速度也会增加，同时蝙蝠攻击力也会增强
